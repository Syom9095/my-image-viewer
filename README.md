# TypeScriptとElectronで作成した、シンプルで高速な画像ビューワー

フォルダ内の画像を矢印キーで軽快に切り替えることができる。

## ファイル構成
main.ts	　　    アプリのウィンドウを作り、OS（Windows）と通信してフォルダ選択ダイアログを表示する。
renderer.ts	    画面の表示やボタンのクリック、矢印キーの入力を処理する。
index.html	    画面の見た目。画像を表示する場所や、フォルダを選択するボタンを配置している。
package.json	プロジェクトの設定図。使っているライブラリ（Electronなど）や、起動コマンドが書かれている。
dist/	            実行用フォルダ。TypeScriptをパソコンが理解できるJavaScriptに変換（コンパイル）したファイルが入る。
start-viewer.bat	起動用スイッチ。ダブルクリックするだけでアプリを立ち上げるためのファイル。

### 実行手順

1.開発・修正時、コード（.tsファイル）を書き換えたら、必ず以下のコマンドをターミナルで実行する。
#### メインプロセスの変換
npx tsc main.ts --outDir dist --target es6 --module commonjs

#### レンダラープロセスの変換
npx tsc renderer.ts --outDir dist --target es6 --module none --skipLibCheck

2.以下のいずれかの方法で起動
・VS Codeから: ターミナルで npm start と入力。
・フォルダから: start-viewer.bat をダブルクリック。

### 使用方法
1.アプリが立ち上がったら 「フォルダを選択する」 ボタンを押す。
見たい画像が入っているフォルダを選んで「フォルダの選択」を押す。

画像が表示されたら、キーボードの 「➡（右矢印）」 で次の画像、 「⬅（左矢印）」 で前の画像に切り替わる。
